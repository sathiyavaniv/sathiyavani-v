Mallcom India – Financial Performance Analysis (Power BI)

Project Overview
This project presents a comprehensive financial performance analysis of Mallcom (India) Ltd
using historical financial statement data. The analysis focuses on evaluating revenue growth,
profitability, margins, and overall financial health through interactive Power BI dashboards.

The project is designed from a Financial Analyst and Business Analyst perspective, emphasizing
trend analysis, KPI tracking, and data-driven insights for decision-making.

--------------------------------------------------

Tools and Technologies
Power BI – Data modeling, DAX measures, and dashboard creation  
Excel – Data cleaning and financial data preparation    
GitHub – Documentation and project version control  

--------------------------------------------------

Data Sources
The analysis is based on real-world financial data collected from multiple reliable sources:

Screener.in  
Used for consolidated financial statements including income statement data such as revenue,
operating profit, net profit, and margins.

Annual Reports of Mallcom (India) Ltd  
Used to cross-verify financial figures, understand business performance, and validate trends.

Company Filings and Public Financial Disclosures  
Used to ensure accuracy and consistency of reported financial numbers.

Excel-based Financial Dataset  
Prepared from the above sources and used as the direct input file for Power BI.

--------------------------------------------------

Dashboard Coverage
The Power BI dashboard includes the following analytical views:

Financial Overview  
Total Revenue  
Net Profit  
Operating Margin Percentage  
Net Profit Margin Percentage  

Trend Analysis  
Revenue trend across multiple years  
Net profit growth trend  
Year-on-Year financial performance comparison  

Profitability Analysis  
Operating profit versus net profit comparison  
Margin stability and efficiency analysis  
Identification of growth and decline periods
